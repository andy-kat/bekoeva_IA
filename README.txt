Movie Library — приложение для управления коллекцией фильмов
Автор: Бекоева Анастасия Вячеславовна

Описание
Movie Library — это простое приложение на Python с графическим интерфейсом, созданное с помощью библиотеки tkinter. Оно позволяет вести базу фильмов, добавлять новые записи, фильтровать их по жанру и году выпуска, а также сохранять данные в файл формата JSON.

Возможности
Добавление новых фильмов с указанием названия, жанра, года выпуска и рейтинга.
Фильтрация фильмов по жанру и/или году.
Сохранение и загрузка базы данных из файла movies.json.
Удобный просмотр фильмов в виде таблицы.
Требования
Python 3.x
Библиотека tkinter (входит в стандартную поставку Python)
Установка и запуск
Убедитесь, что у вас установлен Python 3.
Скачайте или клонируйте репозиторий с проектом.
Запустите файл movie_library.py:
bash
Копировать
python movie_library.py
Использование
Добавление фильма
Введите название, выберите жанр, укажите год выпуска и рейтинг (от 1 до 10).
Нажмите «Добавить фильм».
Фильтрация
Выберите жанр из выпадающего списка или введите год.
Нажмите «Применить фильтр».
Сброс фильтра
Нажмите «Сбросить фильтр», чтобы показать все фильмы.
Сохранение
Все изменения автоматически сохраняются в файл movies.json при добавлении нового фильма.
Примечания
Приложение не требует дополнительной настройки.
Данные хранятся в файле movies.json в той же папке, что и программа.
Если файл данных отсутствует, он будет создан автоматически при первом запуске.
