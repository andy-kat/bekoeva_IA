Цитата дня (Quote of the Day)
Автор: Бекоева Анастасия Вячеславовна
Простое приложение с графическим интерфейсом для генерации, сохранения и управления случайными цитатами. Программа получает цитаты через публичный Forismatic API, позволяет сохранять их в список, фильтровать и удалять.

Как запустить
Убедитесь, что у вас установлен Python 3.
Установите необходимую библиотеку requests:
bash
Копировать
pip install requests
Сохраните код в файл (например, quote_app.py).
Запустите программу из терминала:
bash
Копировать
python quote_app.py
Как пользоваться
Генерация и сохранение цитаты
Нажмите кнопку «Сгенерировать цитату».
Приложение запросит случайную цитату на русском языке через интернет.
Цитата автоматически добавится в список ниже и сохранится в файл tasks.json.
Управление списком
Удалить выбранную: Выберите одну цитату в списке и нажмите эту кнопку, чтобы удалить её.
Показать все: Отменяет все фильтры и отображает полный список сохраненных цитат.
Фильтрация цитат
Введите часть имени автора или темы в соответствующие поля.
Тема определяется автоматически как первое слово из текста цитаты.
Нажмите кнопку «Фильтровать».
В списке останутся только те цитаты, которые соответствуют обоим условиям (И автор, И тема). Если поле пустое, оно не учитывается при фильтрации.
Технические детали
Хранение данных: Все добавленные вами цитаты сохраняются в файл tasks.json в той же папке, что и программа. При следующем запуске они будут загружены автоматически.
API: Для получения цитат используется публичный сервис Forismatic. Для работы программы требуется подключение к интернету.
.gitignore: Для этого проекта рекомендуется создать файл .gitignore и добавить в него строку tasks.json, чтобы не загружать личные данные (ваши любимые цитаты) в систему контроля версий.
